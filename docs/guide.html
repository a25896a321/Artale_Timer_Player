⏱ Artale Timer Player

作者：oo_jump　｜　計算最佳互動時間的小工具

1. 簡介

Artale Timer Player 是一款 Windows 桌面工具， 協助玩家在基礎時間未知的情況下，計算出最適合的互動按鈕組合， 讓計時器盡可能接近最大值 11:50，同時避免超過 12:00。

💡 基礎時間範圍：0:10 ～ 11:50，以 10 分鐘為單位。
目標是透過增時操作，讓最終時間盡量接近 11:50 但不達 12:00。

2. 快速開始

安裝依賴

pip install Pillow

執行程式

python main.py

若要啟用全域快捷鍵，請以系統管理員身份執行（右鍵 → 以系統管理員身份執行）。 即使不是管理員，本工具仍可透過 VK 輪詢監聽熱鍵，無需 RegisterHotKey。

  1. 從 時間按鈕區（10分 / 30分 / 50分 / 1hr / 2hr / 4hr / 9hr / x2）點選最多 4 個按鈕。
  2. 已選按鈕顯示於 預覽區，點擊預覽槽可退回一個選項。
  3. 選滿即自動計算；選 1～3 個亦可立即查看建議結果。
  4. 計算結果按不同排序方式列出各基礎時間段的最佳操作。

5. 懸浮框模式

點擊頂部工具列的 懸浮框 按鈕即可開啟。懸浮框為無邊框視窗，具備：

  • 左上角 ✥ 移動手柄：拖曳移動位置
  • ⚙ 齒輪圖示：點擊返回主視窗設定
  • ✕ 關閉按鈕
  • 右下角 ◢ 拖拽縮放：調整大小並自動儲存
  • 預覽槽：顯示已選按鈕，點擊可退回
  • 時間按鈕：可在介面設定中開關
  • 計算結果：可選「展開模式（預設）」或「滾動模式」
懸浮框設定（大小、位置、顏色、透明度等）可在 🖥 介面設定 → 懸浮框 頁籤調整，儲存後立即刷新。

6. 介面設定

🗂 主視窗 Tab

設定預覽區的文字顏色、背景；主視窗時間按鈕的顏色、背景、快捷鍵提示顯示、按鈕大小與間距。

🗂 懸浮框 Tab

設定懸浮框的背景色、透明度、字體大小與顏色、寬高、起始座標、是否顯示時間按鈕、計算結果顯示方式（展開／滾動）；懸浮框按鈕顏色獨立控制。

🗂 結果顯示 Tab

設定結果字體大小與顏色；開關「顯示編號」、「顯示推估值」、「顯示（任意）提示」、「顯示（依序）提示」；選擇排列方式。

所有顏色設定均提供 選色（Pick） 按鈕，點擊即可開啟系統色彩選擇器。

7. 快捷鍵設定

點擊 💾 快捷鍵 開啟設定視窗。點擊任一操作對應的按鈕，然後按下您想指定的實體按鍵，系統將自動偵測（使用 GetAsyncKeyState，無需管理員權限）。

使用的是 VK Code 輪詢方案，可精確區分數字鍵盤與主鍵盤的相同數字鍵。

8. 計算邏輯

核心演算法對所選按鈕的所有非空子集與排列組合進行窮舉，找出每個基礎時間下能最大化最終時間（且不超過 11:59）的最優序列。

  1. 列舉每個基礎時間（10 ～ 710 分鐘，步距 10 分鐘）。
  2. 對所有子集和排列組合模擬，找出最佳結果序列。
  3. 將連續基礎時間且最佳序列相同的組合合併為一個區間。
  4. 依選定的排序方式輸出最終建議。

由於 x2 是乘法，執行順序不同結果也不同；純加法操作則視為可交換（任意順序）。

10. 常見問題

Q：快捷鍵無效？

確認程式視窗有聚焦，或以管理員身份執行以啟用全域模式（VK 輪詢本身無需管理員，但系統安全軟體可能干擾）。

Q：懸浮框的圖示顯示為文字（✥ ⚙）？

請將 Set_Arrow_keys.png 和 Set_gear.png 放入 png_type/ 資料夾，並安裝 Pillow。

Q：設定儲存在哪裡？

儲存於程式同目錄的 settings.json。刪除此檔案可恢復所有預設值。

Q：如何切換語言？

點擊頂部的 EN / 中文 按鈕，語言即時切換，無需重啟。
